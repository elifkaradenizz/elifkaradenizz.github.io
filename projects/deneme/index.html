Proje Özeti

Project Overview

Müşteri verileri analiz edilerek segmentler oluşturulur ve öneri modelleri geliştirilir. Dashboard interaktif olarak sunulur.

Customer data is analyzed to create segments and generate recommendation models. The dashboard is presented interactively.

  • Performans görselleştirmesi
  • Model sonuçlarının dashboard'da gösterimi
  • CSV upload and preprocessing
  • K-Means based segmentation
  • Performance visualization
