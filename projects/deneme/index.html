Proje Özeti

Project Overview

Bu proje, e‑ticaret müşteri verilerini analiz ederek anlamlı segmentler oluşturur, ardından bu segmentlere göre kişiselleştirilmiş öneri modelleri üretir. Segmentasyon süreci etkileşimli arayüz ile kullanıcıya sunulur.

This project analyzes e-commerce customer data to create meaningful segments and generates personalized recommendation models for each segment. The segmentation process is presented through an interactive interface.

Özellikler

Features

  • CSV dosyası yükleme ve ön işleme
  • K‑Means tabanlı müşteri segmentasyonu
  • Performans metrikleri ve görselleştirme
  • Öneri model sonuçlarının görüntülenmesi
  • CSV upload and preprocessing
  • K-Means based customer segmentation
  • Performance metrics and visualization
  • Recommendation model outputs display

Mimari ve Akış

System Architecture

Sistem; veri ön işleme → kümeleme → modelleme → sonuç gösterimi adımlarından oluşur. Arayüz, bu akışı görsel olarak destekler.

The system consists of data preprocessing → clustering → modeling → output layers. The interface visually supports this workflow.
