LLM File Analysis Chatbot LLM Dosya Analiz Chatbotu

AI-powered chatbot that analyzes CSV, Excel and PDF files using Large Language Models (LLM) Büyük Dil Modelleri (LLM) kullanarak CSV, Excel ve PDF dosyalarını analiz eden yapay zeka chatbot sistemi

This project enables users to analyze structured and unstructured data files without any technical knowledge. The system reads CSV, Excel and PDF files, processes the content, and generates contextual answers using LLMs. Users can interact with their data through natural language queries or use the system as a general AI assistant. Bu proje, kullanıcıların teknik bilgiye ihtiyaç duymadan CSV, Excel ve PDF gibi veri dosyalarını analiz etmesini sağlar. Sistem dosya içeriğini okuyarak LLM (Büyük Dil Modeli) ile bağlamsal cevaplar üretir. Kullanıcılar doğal dil ile veri sorgulaması yapabilir veya sistemi genel amaçlı yapay zeka asistanı olarak kullanabilir.

Response GenerationCevap Üretimi
from langchain.chat_models import ChatOpenAI
import pandas as pd
import PyPDF2

df = pd.read_csv("data.csv")

llm = ChatOpenAI(model="gpt-4")

question = "What is the highest selling product?"

response = llm.predict(
    f"""
    Data:
    {df.head().to_string()}

    Question:
    {question}
    """
)

print(response)
