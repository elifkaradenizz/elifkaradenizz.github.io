LLM File Analysis Chatbot LLM Dosya Analiz Chatbotu

AI-powered chatbot that analyzes CSV, Excel and PDF files using Large Language Models (LLM) Büyük Dil Modelleri (LLM) kullanarak CSV, Excel ve PDF dosyalarını analiz eden yapay zeka destekli sohbet botu

Project Type Proje Türü

Machine Learning / Data Analysis Makine Öğrenmesi / Veri Analizi

Core Technology Temel Teknoloji

OpenAI GPT (LLM) OpenAI GPT (Büyük Dil Modeli)

File Types Dosya Türleri

CSV / Excel / PDF CSV / Excel / PDF

Status Durum

Completed Tamamlandı

This project enables users to analyze structured and unstructured data files without any technical knowledge. The system reads CSV, Excel and PDF files, processes the content, and generates contextual answers using LLMs. Users can interact with their data through natural language queries or use the system as a general AI assistant. Bu proje, kullanıcıların herhangi bir teknik bilgiye sahip olmadan yapılandırılmış ve yapılandırılmamış veri dosyalarını analiz etmelerine olanak tanır. Sistem, CSV, Excel ve PDF dosyalarını okur, içeriği işler ve büyük dil modelleri (LLM) kullanarak bağlamsal yanıtlar üretir. Kullanıcılar, doğal dil sorguları yoluyla verileriyle etkileşime girebilir veya sistemi genel bir yapay zeka asistanı olarak kullanabilir.
