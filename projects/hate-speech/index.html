Status

This project focuses on detecting hate speech and sexist language in social media using Natural Language Processing techniques. A complete machine learning pipeline was developed including text preprocessing, TF-IDF feature extraction, and classification models. The final model achieved 91% accuracy and 0.89 F1-score, showing strong performance on noisy and imbalanced real-world datasets. The system contributes to ethical AI by supporting automated moderation and reducing harmful online content.
