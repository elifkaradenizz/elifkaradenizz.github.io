This project focuses on detecting hate speech and sexist language in social media using Natural Language Processing techniques. A full pipeline including text preprocessing, TF-IDF feature extraction, and classification models was implemented. Different machine learning algorithms were tested and evaluated to improve performance. Bu proje, sosyal medyada nefret söylemi ve cinsiyetçi ifadeleri tespit etmek amacıyla Doğal Dil İşleme tekniklerini kullanmaktadır. Metin ön işleme, TF-IDF özellik çıkarımı ve sınıflandırma modellerini içeren uçtan uca bir pipeline geliştirilmiştir. Farklı makine öğrenmesi algoritmaları test edilerek performans karşılaştırması yapılmıştır.
