E-commerce Recommendation System E-Ticaret Öneri Sistemi
Python Python ML Makine Öğrenmesi
I developed an effective customer recommendation system using Python and various machine learning algorithms. The system analyzes customer behaviors, segments them into meaningful groups, and provides personalized product or service recommendations. It helps optimize marketing strategies, increase engagement, and create targeted campaigns for different customer segments. Python ve çeşitli makine öğrenimi algoritmaları kullanarak etkili bir müşteri öneri sistemi geliştirdim. Sistem, müşteri davranışlarını analiz eder, bunları anlamlı gruplara ayırır ve kişiselleştirilmiş ürün veya hizmet önerileri sunar. Bu sistem, pazarlama stratejilerinin optimize edilmesine, müşteri etkileşiminin artırılmasına ve farklı müşteri segmentleri için hedefli kampanyalar oluşturulmasına yardımcı olur..

Hate Speech Detection Nefret Söylemi Tespiti
NLP NLP Text Classification Metin Sınıflandırma
In this project, NLP techniques were applied to detect and analyze sexism and hate speech on social media. Tokenization, lemmatization, and stop-word removal were performed using NLTK, and texts were vectorized with Word2Vec. WordCloud visualizations highlighted frequent terms, while multiple machine learning algorithms classified harmful content.
Our team presented this work at the 7th International Congress on Multidisciplinary Scientific Studies in Kyrenia, TRNC Bu projede, sosyal medyadaki cinsiyetçilik ve nefret söylemini tespit etmek ve analiz etmek amacıyla NLP teknikleri uygulanmıştır. NLTK kullanılarak tokenleştirme, kök sözcükleştirme ve durdurma sözcüklerinin çıkarılması işlemleri gerçekleştirilmiş, metinler ise Word2Vec ile vektörleştirilmiştir. WordCloud görselleştirmeleri sık kullanılan terimleri öne çıkarırken, çeşitli makine öğrenimi algoritmaları zararlı içeriği sınıflandırmıştır.
Ekibimiz bu çalışmayı KKTC'nin Girne kentinde düzenlenen 7. Uluslararası Disiplinlerarası Bilimsel Çalışmalar Kongresi'nde sundu.

Carbon Emission Prediction Karbon Emisyon Tahmini
Deep Learning Derin Öğrenme CNN CNN
Implemented a deep learning system to predict carbon emissions using DenseNet, NASNet, and Xception models. The system processes industrial and environmental data to forecast emissions, helping organizations make sustainable decisions. Models were evaluated with real-world datasets for accuracy.. DenseNet, NASNet ve Xception modellerini kullanarak karbon emisyonlarını tahmin eden bir derin öğrenme sistemi geliştirdim. Sistem, endüstriyel ve çevresel verileri işleyerek emisyon tahminleri sunar ve kuruluşların sürdürülebilir kararlar almasına yardımcı olur. Modeller, gerçek dünya verileri ile doğruluk açısından değerlendirildi.

LLM LLM Streamlit Streamlit

Image Processing Streamlit Dashboard Görüntü İşleme Streamlit Kontrol Paneli
Deep Learning Derin Öğrenme CNN CNN
Developed an interactive image processing application using Streamlit, allowing users to apply filters, segmentations, and transformations. Techniques such as histogram equalization, morphological operations, and pixel-level analysis are implemented for educational and practical purposes. Kullanıcıların filtreler, segmentasyonlar ve dönüşümler uygulayabileceği etkileşimli bir görüntü işleme uygulaması geliştirdim. Histogram eşitleme, morfolojik işlemler ve piksel düzeyinde analiz gibi teknikler, hem eğitim hem de pratik amaçlarla uygulanmıştır.
