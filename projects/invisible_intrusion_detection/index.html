Algorithm Algoritma

Status Durum

Completed Tamamlandı

This project detects network intrusions using unsupervised learning techniques. The main goal is maximizing recall for attack detection without labeled data. Bu proje, denetimsiz öğrenme yöntemleri kullanarak ağ trafiğinde saldırı tespiti yapmayı amaçlar. Temel hedef etiket bilgisi olmadan saldırıları yüksek recall ile yakalamaktır.
