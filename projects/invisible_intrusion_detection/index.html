This project detects network intrusions using unsupervised learning techniques. The main goal is maximizing recall for attack detection without labeled data. Bu proje, denetimsiz öğrenme yöntemleri kullanarak ağ trafiğinde saldırı tespiti yapmayı amaçlar. Temel hedef etiket bilgisi olmadan saldırıları yüksek recall ile yakalamaktır.

Key Findings Önemli Bulgular

  • In this project, accurately detecting attacks takes precedence over general accuracy metrics. For this reason, recall has been treated as the most critical evaluation metric. Bu projede, saldırıları doğru bir şekilde tespit etmek genel doğruluk ölçütlerinden daha önceliklidir. Bu nedenle, recall en önemli değerlendirme ölçütü olarak kabul edilmiştir.
  • Isolation Forest has demonstrated better performance in recall-based evaluations Isolation Forest, geri çağırma tabanlı değerlendirmelerde daha iyi bir performans sergilemiştir
