This project combines traditional NLP machine learning techniques with Large Language Model (LLM) capabilities. It provides sentiment analysis using classical ML models and enriches outputs with LLM-based explanations, content rewriting, hashtag generation and user engagement insights. Bu proje, klasik NLP makine öğrenmesi tekniklerini büyük dil modeli (LLM) yetenekleri ile birleştirir. Duygu analizi klasik ML modelleri ile yapılırken, LLM ile açıklama üretimi, metin yeniden yazımı, hashtag üretimi ve etkileşim odaklı içgörüler sağlanır.
