Intrusion Detection System (UNSW-NB15) Siber Saldırı Tespit Sistemi (UNSW-NB15)

Unsupervised machine learning based intrusion detection system using DBSCAN and Isolation Forest algorithms DBSCAN ve Isolation Forest algoritmaları kullanılarak geliştirilmiş denetimsiz öğrenme tabanlı siber saldırı tespit sistemi

Main Algorithms Ana Algoritmalar

DBSCAN / Isolation Forest / PCA DBSCAN / Isolation Forest / PCA

Status Durum

Completed Tamamlandı

This project focuses on detecting network intrusions and anomalies using unsupervised learning techniques. DBSCAN and Isolation Forest algorithms were applied and compared on the UNSW-NB15 dataset. The main goal is to detect attack patterns without using labeled data and maximize recall for intrusion detection. Bu proje, denetimsiz öğrenme yöntemleri kullanarak ağ trafiğinde saldırı ve anomali tespiti yapmayı amaçlar. UNSW-NB15 veri seti üzerinde DBSCAN ve Isolation Forest algoritmaları karşılaştırmalı olarak uygulanmıştır. Temel hedef, etiket bilgisi kullanmadan saldırıları tespit etmek ve özellikle recall değerini maksimum seviyeye çıkarmaktır.
